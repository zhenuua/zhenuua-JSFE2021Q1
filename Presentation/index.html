Electron Advantages

  • Building desktop apps with web technologies
  • Low barrier to entry
  • Ease with finding developers
  • Cross Platform
  • Use whatever tools and web frameworks/libraries
  • Debugging the renderer
  • Your web code can access native system APIs and pretty much anything else on the computer
  • Auto Update
  • Your code and related content can be local to your app or it can be served remotely from a server similar to a hybrid app

Electron Disadvantages

  • Slow start / slow running (Compared to native executables)
  • High Memory and Storage Usage
  • Security Challenges
